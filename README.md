# ResearchBuddy

🔬 陪你做科研的 AI 伙伴 — 记录、回顾、鼓励、打卡、交流

## 功能模块

- 📝 **科研日记** — 每日记录科研进展
- 🤖 **AI 陪伴** — 基于日记内容对话，情绪识别，提供支持和建议
- 📊 **进度追踪** — 可视化科研进度，里程碑管理
- 🔄 **回顾系统** — 周/月总结，对比历史，发现模式
- ✅ **自律打卡** — 每日科研习惯打卡，streak 激励
- 💬 **社群交流** — 科研人员匿名/实名交流区

## 技术栈

- Next.js 14 + TypeScript
- Tailwind CSS
- Prisma + SQLite
- OpenAI / Claude API
- NextAuth.js

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 浏览器打开 http://localhost:3000
```
